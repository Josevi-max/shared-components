# SharedComponents

Librería de Web Components reutilizables creada con **Lit** y **Vite**, lista para consumir en proyectos Angular, vanilla JS u otros frameworks.

Esta librería sigue **Atomic Design**:

- **Atoms:** componentes básicos (botones, inputs)
- **Molecules:** combinaciones de atoms (search bar)
- **Organisms:** componentes de mayor nivel (user card, card lists)

---

## Instalación

Desde otro proyecto, instala la librería local o desde npm si está publicada:

```bash
npm install ../SharedComponents
